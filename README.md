# JPMC interview challenge

JPMC interview challenge developed in React.js. This app uses version 13.8.0 of Node.js.

**1.** Clone the repo:

```
$ git clone git@github.com:gicanas94/jpmc-challenge.git
$ cd jpmc-challenge
```

**2.** Install dependencies:

```
$ npm install
```

**3.** Run the server:

```
$ npm run server
```

**4.** Run the app:

```
$ npm run app
```
